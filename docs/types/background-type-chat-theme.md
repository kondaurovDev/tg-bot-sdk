# BackgroundTypeChatTheme

The background is taken directly from a built-in chat theme.

[Telegram docs](https://core.telegram.org/bots/api#backgroundtypechattheme)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| type | `"chat_theme"` | Yes | Type of the background, always “chat_theme” |
| theme_name | `string` | Yes | Name of the chat theme, which is usually an emoji |
