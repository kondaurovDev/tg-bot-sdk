# Birthdate

Describes the birthdate of a user.

[Telegram docs](https://core.telegram.org/bots/api#birthdate)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| day | `number` | Yes | Day of the user&#39;s birth; 1-31 |
| month | `number` | Yes | Month of the user&#39;s birth; 1-12 |
| year | `number` | No | Year of the user&#39;s birth |
