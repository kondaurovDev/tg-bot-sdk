# PaidMediaPurchased

This object contains information about a paid media purchase.

[Telegram docs](https://core.telegram.org/bots/api#paidmediapurchased)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| from | [`User`](user.md) | Yes | User who purchased the media |
| paid_media_payload | `string` | Yes | Bot-specified paid media payload |
