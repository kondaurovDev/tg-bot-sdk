# PaidMediaPreview

The paid media isn&#39;t available before the payment.

[Telegram docs](https://core.telegram.org/bots/api#paidmediapreview)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| type | `"preview"` | Yes | Type of the paid media, always “preview” |
| width | `number` | No | Media width as defined by the sender |
| height | `number` | No | Media height as defined by the sender |
| duration | `number` | No | Duration of the media in seconds as defined by the sender |
