# LocationAddress

Describes the physical address of a location.

[Telegram docs](https://core.telegram.org/bots/api#locationaddress)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| country_code | `string` | Yes | The two-letter ISO 3166-1 alpha-2 country code of the country where the location is located |
| state | `string` | No | State of the location |
| city | `string` | No | City of the location |
| street | `string` | No | Street address of the location |
