# PaidMessagePriceChanged

Describes a service message about a change in the price of paid messages within a chat.

[Telegram docs](https://core.telegram.org/bots/api#paidmessagepricechanged)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| paid_message_star_count | `number` | Yes | The new number of Telegram Stars that must be paid by non-administrator users of the supergroup chat for each sent message |
