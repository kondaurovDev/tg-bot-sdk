# UserChatBoosts

[Telegram docs](https://core.telegram.org/bots/api#userchatboosts)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| boosts | [`ChatBoost`](chat-boost.md)[] | Yes | The list of boosts added to the chat by the user |
