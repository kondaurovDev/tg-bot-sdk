# closeGeneralForumTopic

Use this method to close an open &#39;General&#39; topic in a forum supergroup chat The bot must be an administrator in the chat for this to work and must have the can_manage_topics administrator rights

[Telegram docs](https://core.telegram.org/bots/api#closegeneralforumtopic)

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| chat_id | `number` \| `string` | Yes | Unique identifier for the target chat or username of the target supergroup (in the format @supergroupusername) |

## Return type

`boolean`
