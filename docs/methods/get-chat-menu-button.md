# getChatMenuButton

Use this method to get the current value of the bot&#39;s menu button in a private chat, or the default menu button

[Telegram docs](https://core.telegram.org/bots/api#getchatmenubutton)

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| chat_id | `number` | No | Unique identifier for the target private chat If not specified, default bot&#39;s menu button will be returned |

## Return type

[`MenuButton`](../types/menu-button.md)
